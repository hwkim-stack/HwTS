# 🐣 미미(Mimi) — 자동 수익 루프 플레이북

> "돈은 지루한 일을 꾸준히 할 때 벌리고, 이제 그 지루한 일을 AI에게 시킬 수 있다."

이 프로젝트는 그 문장을 **웹 + 무(無)API** 버전으로 옮긴 것이다.
미미-seed 가 *모바일 앱*의 운영을 자동화한다면, 여기서는 *AdSense 유틸리티 사이트*를 같은 루프로 굴린다.

## 솔직한 돈 계산 (먼저 읽기)
광고 수익은 대략:
```
수익 ≈ 도구 수 × 도구별 방문수 × 광고 노출 × RPM
```
- 스크립트를 돌린다고 돈이 생기지 않는다. **검색 트래픽**이 핵심이다.
- 레버리지는 "공짜에 가까운 비용으로 **작은 도구를 계속 늘리는 것**". 한 개가 대박 안 나도, 20~50개가 롱테일로 쌓이면 합산이 의미 있어진다.
- "하루 치킨 3~4마리(약 ₩66,000~88,000/일)"는 **시간이 쌓여** 나오는 결과다. 목표일 뿐 보장이 아니다.
- 현실 타임라인: 색인·승인·트래픽 누적까지 보통 **수 주~수 개월**. 초반 P&L은 마이너스(도메인비)에서 시작한다 — 정상이다.

## 루프 한 바퀴
```
/mimi-plan   기획     다음 도구 1개 결정
   ↓
/mimi-build  개발     도구 구현 + 빌드
   ↓
/mimi-ship   운영     배포 + sitemap/AdSense
   ↓
(며칠 대기 — 트래픽이 쌓이는 동안)
   ↓
/mimi-analyze 분석    metrics.csv 갱신 + 해석
   ↓
/mimi-decide  의사결정 유지·개선·확장·중단 → 다음 기획 방향
   ↺ 다시 /mimi-plan
```
지금 어디인지 모르면 `/mimi-loop` 또는 `/mimi-status`.

## 처음 세팅 (1회)
1. **로컬 확인**: `npm run dev` → http://localhost:8080
2. **도메인 + 호스팅** (택1, 모두 무료 티어 가능):
   - GitHub Pages / Vercel / Cloudflare Pages / Netlify (자세한 명령은 `/mimi-ship`)
   - `mimi/config.json` 의 `baseUrl` 을 실제 도메인으로.
3. **AdSense**:
   - 사이트에 도구가 어느 정도(예: 8~15개) 쌓이고 트래픽이 들어오기 시작하면 AdSense 신청.
   - 승인되면 `mimi/config.json` 의 `adsense.publisherId` 와 `slots` 를 채우고 `npm run build`.
   - 승인 전까지 광고 자리에는 회색 플레이스홀더가 보인다(정상).
4. **Search Console**: 사이트 등록 후 `sitemap.xml` 제출(색인 가속).

## 매일/매주 루틴 (지루한 일 = AI에게)
- 새 도구 1개: `/mimi-plan` → `/mimi-build` → `/mimi-ship`
- 주 1회: `/mimi-analyze` → `/mimi-decide` 로 무엇을 더 만들지 데이터로 결정
- `npm run report` 로 치킨 지표 확인 🍗

## 왜 API 키가 없나
판단(기획·카피·분석)은 **네가 Claude Code 구독으로 직접** `/` 명령에서 돌린다. 빌드·배포·집계 같은 기계적인 일만 스크립트가 한다. → 추가 API 비용 0원.
