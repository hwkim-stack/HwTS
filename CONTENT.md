# 🎬 미미 v2 — AI 콘텐츠 공장 플레이북

> SEO(권위 누적, 무명에게 닫힘) 대신 **알고리즘 분배**(무명에게 열림)를 쓴다.
> 지루한 제작은 AI에게, 판단(니치·훅·품질)은 사람이. 영상은 **복리로 쌓이는 자산**.

## 핵심 원칙
1. **분배 우선.** 기능/완성도보다 "알고리즘이 미느냐(조회·평균시청)"가 전부.
2. **양질만 생존.** 2026년 플랫폼은 저품질 AI 양산물을 수익정지시킨다. AI는 *보조*, 품질 합격선은 사람이 지킨다. "AI 쓰레기 양산"은 죽는 길.
3. **데이터로 니치 선택.** 우위가 없으니 머리로 고르지 말고, 싸게 여러 개 던지고 데이터로 더블다운.
4. **광고 푼돈에 의존 안 함.** 숏폼 광고단가는 매우 낮다. 수익은 ①롱폼 복리 광고 ②제휴 ③디지털상품으로 만든다.

## 루프 (기획→제작→발행→분석→결정)
| 단계 | 명령 | 하는 일 |
|---|---|---|
| 기획 | `/content-plan` | 니치별 영상 아이디어 + 훅 결정 |
| 제작 | `/content-make` | 대본·샷리스트·성우대본·자막·제목·설명·해시태그·썸네일 문구 생성 |
| 발행 | `/content-ship` | 업로드 체크리스트 + 메타데이터, 발행 기록 |
| 분석 | `/content-analyze` | metrics.csv 갱신 + 알고리즘이 보상한 것 해석 |
| 결정 | `/content-decide` | 니치 유지·개선·확장·중단 → 다음 기획 |
현재 위치: `/content-loop` · 대시보드: `/content-status` 또는 `npm run content:report`

## 지금의 실험: 니치 3개 테스트
| id | 니치 | 메커니즘 | 수익화 |
|---|---|---|---|
| money | 1분 금융/세금 상식 | 고RPM(광고) | 광고 + 제휴 |
| mystery | 1분 미스터리/실화 나레이션 | 고리텐션 | 광고 |
| tools | 무료·AI 도구 1분 소개 | 트렌드+제휴 | 제휴 + 광고 |

각 니치 3편씩(총 9편)을 2~3주에 걸쳐 올리고, **조회수·평균시청률**로 이긴 니치에 집중한다.
(니치를 바꾸고 싶으면 `content/state.json` 의 `experiment.niches` 수정 또는 `/content-plan`)

## 영상 품질 합격선 (숏폼 기준)
- **0~2초 훅**: 첫 문장에서 "왜 봐야 하는지"가 꽂혀야 함. 없으면 폐기.
- **길이**: 25~45초. 군더더기 0.
- **자막 必**: 무음 시청자 다수. 큰 자막 + 명확한 발음 TTS.
- **루프/리텐션**: 마지막이 처음과 이어지거나, 끝까지 봐야 답이 나오게.
- **CTA**: 과하지 않게 1개(구독 or 다음 영상).

## 제작 도구
- 이 환경의 MCP: `generate_image` / `generate_video` / `generate_audio`(TTS) / `virality_predictor`(올리기 전 바이럴 점수 체크). **크레딧 비용 가능.**
- 무료 대안: 직접 편집(CapCut 등) + 무료 TTS + 무료 스톡(Pexels/Pixabay). 초기엔 무료 위주로 비용 0 유지 권장.

## 솔직한 기대치
- **0~6주**: 거의 0원. 니치/훅 탐색.
- **6주~3개월**: 한 포맷이 걸리면 우상향 시작. 유튜브 수익화 자격(구독 1천 + 쇼츠 1천만뷰/90일 또는 시청 4천시간) 도전.
- **3~6개월+**: 라이브러리·제휴 복리.
- **확률**: 일찍 그만두거나 니치 잘못 잡으면 대부분 0. "되는" 조건 = 꾸준함 + 데이터 반복. 단, SEO와 달리 **노출은 실제로 일어난다.**
