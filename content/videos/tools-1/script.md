5. "셋째, PDF 다 되는 사이트."
6. "합치기·쪼개기·압축·변환까지 전부 무료. 설치도 가입도 필요 없어요."
7. "이 셋만 알아도 웬만한 외주는 직접 끝납니다."

## 마무리 / CTA (1개만)
> "사이트 이름은 고정 댓글에! 이런 꿀팁 더 필요하면 구독."

## 비주얼 / 샷리스트 (faceless · 화면 녹화 추천)
- 0:00 "외주비 ₩0" 타이포 임팩트
- 0:05 배경제거 전/후 비교(사진 → 배경 사라짐) 화면녹화
- 0:16 무료 스톡 사이트 갤러리 스크롤 화면
- 0:25 PDF 합치기 드래그&드롭 화면녹화
- 0:33 "3개 다 무료" + 구독 버튼 모션
- 배경음: 경쾌한 로열티프리. 자막 항상 대형.

## 성우(TTS) 대본 (구어체, 그대로 읽기)
이거 모르고 디자인 외주비 내고 있었으면 손해예요. 전부 무료, 가입도 거의 필요 없는 사이트 세 개.
첫째, 배경 제거. 사진 한 장 올리면 클릭 한 번에 배경이 사라져요. 포토샵 필요 없습니다.
둘째, 무료 고화질 이미지와 영상. 저작권 걱정 없는 스톡 사이트라 썸네일이든 영상이든 그냥 써도 돼요.
셋째, 피디에프 다 되는 사이트. 합치기, 쪼개기, 압축, 변환까지 전부 무료고 설치도 가입도 필요 없어요.
이 셋만 알아도 웬만한 외주는 직접 끝납니다. 사이트 이름은 고정 댓글에! 더 필요하면 구독.
